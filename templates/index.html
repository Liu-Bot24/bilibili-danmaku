bilibili轻量字幕助手

适用于 Chrome 和 Chromium 内核浏览器，可辅助获取用于本站「字幕深度分析」功能的视频字幕。
也可独立使用，在 B 站视频页查看、搜索、复制和下载可用字幕，并支持双击字幕跳转到对应播放时间。

GitHub 仓库 打开 查看源码、README 和本地安装说明。 下载插件包 v{{ plugin_info.version }} 下载后解压，在 Chrome 扩展程序页面加载解压后的文件夹。

安装教程

  1. 下载并解压插件包 点击上方“下载插件包”，把压缩包解压到一个固定目录，后续不要随意移动这个目录。
  2. 打开扩展程序页面 在地址栏输入 chrome://extensions，进入扩展程序页面。
  3. 加载未打包的扩展程序 打开开发者模式，点击“加载未打包的扩展程序”，选择刚才解压出来、包含 manifest.json 的文件夹。
  4. 确认加载成功 加载成功后，扩展程序列表中会出现哔哩哔哩轻量字幕助手。

使用教程

  1. 在视频页查看字幕 打开任意 B 站视频页，等待页面加载完成后，在右侧弹幕区域查看字幕列表。
  2. 下载字幕并跳转到网站 点击“下载 TXT”保存字幕文件，再点击“AI分析”将自动打开本站字幕深度分析页面。
  3. 上传字幕并开始深度分析 向下滑动到字幕深度分析卡片，点击“上传字幕文件”，选择刚才下载的 TXT 字幕文件，然后点击“开始深度分析”。
