# Bilibili 弹幕分析网站模板

![Bilibili 弹幕分析网站预览](assets/github-preview.jpg)

Languages: 简体中文 · [English](README.en.md)

一个用于搭建 Bilibili 弹幕查询、下载、统计图表和 AI 分析网站的 Flask 模板。用户输入 BV 号或 Bilibili 视频链接后，可以获取弹幕数据、下载 CSV/TXT、查看互动分布，并用 OpenAI-compatible 模型生成弹幕内容分析或字幕深度分析。

它适合用来搭建弹幕下载工具、视频复盘工具、内容研究站，或带分享页的轻量级数据分析网站。

## 功能

- 解析 BV 号或 Bilibili 视频链接
- 查询视频基础信息和封面
- 获取弹幕并导出 CSV / TXT
- 展示弹幕密度、词云、发送日期分布和发送时间分布
- 生成 AI 弹幕内容分析
- 上传字幕并生成字幕深度分析
- 保存分析结果并生成分享页链接
- 生成分享二维码和分享卡片
- 支持浏览器本地自定义 OpenAI-compatible API
- 提供运营看板，用于查看访问、下载、分析任务和分享报告数据

## 快速开始

复制配置模板，填入服务器路径和模型服务信息：

```bash
cp config.template.py config.py
cp .env.example .env
```

本地调试：

```bash
python3 app.py
```

生产环境可以使用 uWSGI、Gunicorn 或面板服务托管，并通过 Nginx 反向代理到后端应用。仓库内提供了 uWSGI、Gunicorn 和启动脚本模板，按你的服务器环境复制并修改即可。

## 配置

常用配置项：

| 配置 | 作用 | 建议 |
| --- | --- | --- |
| 项目运行目录 | 后端应用、静态资源和模板所在位置 | 使用服务器上的稳定目录 |
| 日志路径 | 应用日志、访问日志和 worker 日志 | 放在服务器日志目录 |
| 状态数据库 | 保存任务、统计和限流状态 | 使用可写的 SQLite 路径 |
| 私有密钥文件 | 保存维护 token、模型 API Key 和搜索引擎提交 token | 放在服务器私有目录 |
| 模型配置文件 | 覆盖内置模型、fallback 顺序、超时和采样参数 | 放在服务器私有目录 |
| 模型接口 | OpenAI-compatible Chat Completions endpoint、模型名和 token 上限 | 选择稳定且支持 JSON 输出的模型 |

请把 API Key、维护 token 和提交 token 放在服务器环境变量或私有 JSON 文件中，不要暴露在浏览器、公开网页目录或公开仓库里。

## AI 分析

内置分析流程会把弹幕整理成适合模型阅读的样本，并要求模型返回结构化 JSON。你可以配置：

- 主模型和备用模型
- fallback 顺序
- 普通弹幕分析和字幕深度分析的采样上限
- 请求超时和最大输出 token
- 并发任务数量

用户也可以在网页中配置自己的 OpenAI-compatible API。该自定义配置只保存在用户浏览器本地，不会进入服务器队列。

## 运行数据

网站运行后会生成弹幕文件、字幕文件、分享报告、异步任务状态和统计数据。迁移服务器时，如果需要保留历史下载、分享页或运营数据，请一并迁移这些运行数据目录和状态数据库。

## 部署建议

- 给下载、报告、状态数据库和日志目录配置写入权限。
- 在反向代理层启用 HTTPS、合理的请求体大小和超时。
- 将密钥、token 和私有模型配置放在服务器环境变量或私有文件中。
- 为后台 worker 配置独立进程，避免长时间 AI 分析阻塞网页请求。
- 定期备份状态数据库和需要保留的分享报告。
